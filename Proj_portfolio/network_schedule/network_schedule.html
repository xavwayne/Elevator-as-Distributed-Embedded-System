2.
Stuff bit might increase the total number of bits per seconds to send. The fact that clock drift means that if there are no edges, receivers will lose track of bits. Also, periodic edges allow receiver to resynchronize to sender clock. So one of solution is to add stuff bits. Stuff bits are extra bits added to force transitions regardless of data. In best case, we don't need stuff bits. But int worse case, we need to add an opposite-valued stuff bit after every 5 identical bits.
