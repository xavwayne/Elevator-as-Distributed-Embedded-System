This page lists and describes the java files in your elevatorcontrol package.  Update this document to reflect your implementation and add your java files to the elevatorcontrol folder.

Note:  In order for your portfolio to be complete, the code files in the elevatorcontrol folder must compiled and run if copied into the elevatorcontrol package (overwriting any duplicates) of a clean version of the latest required simulator release.

Controller files

  • CarButtonControl.java
  • HallButtonControl.java
  • DoorControl.java
  • DriveControl.java
  • CarPositionControl.java
  • Dispatcher.java
  • LanternControl.java

Translator files

Links and descriptions to all message translators used by the controllers.
  • NewBooleanCanPayloadTranslator.java - 1 byte boolean CAN translator, used by CarButtonControl, DoorControl, HallButtonControl, LanternControl, Utility
  • DesiredFloorCanPayloadTranslator.java - 2 byte desired floor CAN translator, used by CarPositionControl, Dispatcher, DoorControl, DiveControl, HallButtonControl, LanternControl
  • DoorMotorCommandCanPayloadTranslator.java - 1 byte door motor command CAN translator, used by DoorControl
  • NewIntegerCanPayloadTranslator.java - new 4 byte Integer CAN translator, used by CarPositionControl, Dispatcher, DoorControl, DriveControl
  • DriveCommandCanPayloadTranslator.java - 1 byte drive command CAN translator, used by DriveControl

Utility files

  • Utility.java - changed in project 9 , added two new class to store car call and hall call, and related methods to restore pressed car call and hall call to dispatcher
  • Proj7RuntimeMonitor.java - Created in project 7 , used for count waste openings and other issues required in that porject
  • RuntimeRequirementsMonitor.java - Created in project8, updated in porject10 ,used for monitoring new added high level requirements. Now compelted RT6, 7, 8,9,10.
